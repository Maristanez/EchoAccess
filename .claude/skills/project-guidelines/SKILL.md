# EchoAccess Rules

When working on EchoAccess:
- Frontend code is in the `frontend` directory (React/Vite setup).
- Backend code is in the `backend` directory (Python/FastAPI or Python/Flask).
- We use pre-commit hooks, so ensure code is well-formatted.
- Always check the `README.md` and `walkthrough.md` for high-level guidance.
